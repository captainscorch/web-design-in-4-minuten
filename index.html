Web Design in 4 Minuten

von Jeremy Thomas übersetzt von Daniel Schmier

Sagen wir du hast ein Produkt, ein Portfolio, oder bloß eine Idee welche du mit Allen teilen möchtest, auf deiner eigenen Webseite. Bevor du deine Webseite veröffentlichst, willst du das Sie gut aussieht, professionell ist oder zumindest ansehnlich ist.

Was ist die erste Sache an dem du arbeiten musst?

Inhalt

Der Sinn von Design ist es, die Darstellung der Inhalte, auf die es angewendet wird, zu verbessern. Das erscheint logisch, aber da Inhalte ein primäres Element von Webseiten sind, sollte dies nicht erst im Nachhinein
bedacht werden.

Geschriebener Inhalt, wie den Absatz welchen du gerade liest, macht mehr als 90% des Internets aus. Das gestalten dieses Inhalts macht viel aus.

Nehmen wir an, du hast den Inhalt, welchen du veröffentlichen möchtest, bereits fertiggestellt. Anschließend hast du eine leere style.css Datei erstellt, was ist die erste Regel die du schreiben kannst?

Zentrierung

Lange Textzeilen können sehr gestreckt sein und deshalb auch schwer zu lesen. Setzt man ein Limit für die maximale Anzahl an Zeichen pro Zeile verbessert dies gravierend die Lesbarkeit und das Wirken eines langen Absatzes mit vielen Zeilen.

body {
  margin: 0 auto;
  max-width: 50em;
}

Nach dem gestalten der Textblöcke, wie sieht es aus mit der Gestaltung des Textes selbst?

Font Familie

Der Standard-Font in Browsern ist "Times", welcher nicht sehr ansprechend und modern aussieht. Das wechseln zu einem sans-serif (Font ohne Serifen) wie "Helvetica" oder "Arial" kann das Aussehen deiner Seite schnell verbessern.

body {
  font-family: "Helvetica", "Arial", sans-serif;
}

Wenn du trotzdem einen Font mit Serifen verwenden möchtest, versuche "Georgia".

Diese Änderung macht den Text interessanter, machen wir Ihn jetzt noch lesbarer.

Abstände

Wenn eine Seite für Besucher "kaputt" aussieht, handelt es sich meistens um ein Problem mit Abständen. Hinzufügen von Abstand um und innerhalb deines Inhaltes steigert die Attraktivität deiner Seite.

body {
  line-height: 1.5;
  padding: 4em 1em;
}

h2 {
  margin-top: 1em;
  padding-top: 1em;
}

Jetzt wo sich das Layout stark verbessert hat, können wir weitere kleine Änderungen vornehmen.

Farben & Kontrast

Schwarzer Text auf weißem Hintergrund kann für deine Augen anstrengend zum lesen sein. Tendiert man zu einem weicherem Schwarzton für den Fließtext, hilft dieser die Seite angenehmer zum lesen zu machen.

body {
  color: #555;
}

Für einen starken Kontrast, fügen wir einen noch dunkleren Schwarzton für wichtige Wörter hinzu. 

h1,
h2,
strong {
  color: #333;
}

Jetzt, wo ein großteil der Seite bereits visuell aufgebessert wurde, sehen manche Elemente (wie z.B. die Code Snippets) immernoch ungestaltet aus.

Balance

Es braucht nur einige wenige Anpassungen um die Ausgewogenheit der Seite wiederherzustellen:

code,
pre {
  background: #eee;
}

code {
  padding: 2px 4px;
  vertical-align: text-bottom;
}

pre {
  padding: 1em;
}

An diesem Punkt willst du deine Seite vielleicht so gestalten, dass Sie sich aus der Masse hervorhebt.

Primärfarbe

Die meisten Unternehmen haben eine Primärfarbe welche als prägnanter Akzent dient. Auf Webseiten kann dieser Akzent verwendet werden, um interaktive Elemente, wie Links hervorzuheben.

a {
  color: #e81c4f;
}

Um die Balance zu wahren, fügen wir weitere Farben hinzu.

Sekundärfarbe

Die Akzentfarbe kann mit weiteren, feinen Farbtönen komplimentiert werden, welche sich für die Verwendung bei Rändern, Hintergründen oder dem Fließtext eignen.

body {
  color: #566b78;
}

code,
pre {
  background: #f5f7f9;
  border-bottom: 1px solid #d8dee9;
  color: #a7adba;
}

pre {
  border-left: 2px solid #69c;
}

Jetzt wo wir die Farbtöne angepasst haben, wieso nicht auch gleich die Formen...

Eigener Font

Da Text den meisten Inhalt auf einer Webseite darstellt, verhilft ein eigener Font der Seite zu noch mehr eigener Identität.

Du kannst deinen eigenen Webfont oder einen Service wie Typekit für diesen Zwecke verwenden. Wir begnügen uns mit "Roboto", ein Font aus dem kostenlosen Google Fonts Service:

@import 'https://fonts.googleapis.com/css?family=Roboto:300,400,500';

body {
  font-family: "Roboto", "Helvetica", "Arial", sans-serif;
}

Nachdem aufbessern deiner Seiten-Identität mit Text, wie wäre es mit dem hinzufügen von tausend Wörtern...

Grafiken und Icons können entweder als Ornamente benutzt werden um den Inhalt zu unterstützen,
oder um aktiv deine gewünschte Nachricht zu vermitteln.

hübschen wir die Webseite noch schnell mit einem schönen Hintergrundbild von Unsplash auf

header {
  background-color: #263d36;
  background-image: url("header.jpg");
  background-position: center top;
  background-repeat: no-repeat;
  background-size: cover;
  line-height: 1.2;
  padding: 10vw 2em;
  text-align: center;
}

Fügen wir noch ein Logo hinzu

header img {
  display: inline-block;
  height: 120px;
  vertical-align: top;
  width: 120px;
}

Nutzen wir die Möglichkeit zum aufbessern der Text Stile.

header h1 {
  color: white;
  font-size: 2.5em;
  font-weight: 300;
}

header a {
  border: 1px solid #e81c4f;
  border-radius: 290486px;
  color: white;
  font-size: 0.6em;
  letter-spacing: 0.2em;
  padding: 1em 2em;
  text-transform: uppercase;
  text-decoration: none;
  transition: none 200ms ease-out;
  transition-property: color, background;
}

header a:hover {
  background:  #e81c4f;
  color: white;
}

Und voilà!

Wir haben eine ordentliche Seite in nur wenigen Minuten gestaltet, dank des befolgens von simplen Prinzipien des Web Designs. Jetzt gibt es nur noch eine letzte Sache zu erledigen...
